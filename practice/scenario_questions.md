# Pooled scenario questions

This is the pooled timed question set, tagged by source lab and domain. Each question carries at
least one Branch A fabricated-feature distractor and at least two Branch B valid-but-suboptimal
options, per `../DISTRACTOR_TAXONOMY.md`. Answers and distractor labels live in
`answer_rubrics.md`.

Status: seeded for v0.1, not yet filled. Questions are added as each lab is built out, so that no
question ships without its paired lab. See `../WHAT_NOT_TO_DO.md` ("do not build a question bank
only").
