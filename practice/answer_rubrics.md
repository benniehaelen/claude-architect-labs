# Pooled answer rubrics

Explained answers for the pooled timed set in `scenario_questions.md`. Every entry names the
correct option, labels each distractor by branch (A fabricated feature, B valid but suboptimal),
and states the specific reason it fails: the nonexistent switch for Branch A, and the precise
judgment error for Branch B.

Status: seeded for v0.1, not yet filled. Rubrics are added alongside their questions as each lab is
built out.
